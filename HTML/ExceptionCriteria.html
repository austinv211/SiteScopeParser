CRITERIA FOR EXCEPTIONS

CPU

Utilization greater than 70% for sustained periods of time.
Dramatic increases in utilization.
Upward steps in CPU uttdzation equivalent to the capacity of one or more CPUs.

MEMORY

Dramatic drops in free memory.
Steady decline in free memory.
Less than 80% available swap space.
Review of other surrogate memory parameters.

FILE SYSTEM

Dramatic increases in space usage for non-database file systems.
For database file systems, ensure that DBAs are monitoring sub-allocated file systems.
Greater than or equal to 85% utilization Warning threshold.
Less than or equal to 95% utilization Critical threshold.
